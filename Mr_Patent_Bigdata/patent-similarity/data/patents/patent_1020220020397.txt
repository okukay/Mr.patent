(요약)

(청구항)

청구항 1
신발 관리 장치를 제어하는 방법에 있어서,적어도 하나의 센서를 이용하여 신발의 오염도 정보를 획득하는 단계;복수의 동작 모드 중 하나의 동작 모드로 신발 관리 동작을 수행하는 단계;상기 신발의 오염도 정보 또는 상기 수행 중인 동작 모드 중 적어도 하나에 기초하여, 적어도 하나의 로봇 청소기로 청소 요청을 전송할지 여부를 결정하는 단계; 및상기 청소 요청을 전송하는 것으로 결정된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 청소 요청을 전송하는 단계를 포함하는 신발 관리 장치 제어 방법.

청구항 2
제1항에 있어서,상기 신발의 오염도 정보를 획득하는 단계는,먼지 센서를 이용하여 먼지 오염도 정보를 획득하는 단계; 및습기 센서를 이용하여 습기 오염도 정보를 획득하는 단계를 포함하는, 신발 관리 장치 제어 방법.

청구항 3
제2항에 있어서,상기 청소 요청을 전송할지 여부를 결정하는 단계는,상기 먼지 오염도 정보가 먼지 기준 값 이상이라고 판단된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 먼지 청소를 요청하는 것으로 결정하는 단계; 및상기 습기 오염도 정보가 습기 기준 값 이상이라고 판단된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 물걸레 청소를 요청하는 것으로 결정하는 단계를 포함하는, 신발 관리 장치 제어 방법.

청구항 4
제1항에 있어서,상기 복수의 동작 모드는, 신발의 냄새 및 먼지를 제거하고 습기를 제거하는 제1 모드, 젖은 신발의 습기를 제거하는 제2 모드, 및 외출 전 신발을 관리하는 제3 모드를 포함하는, 신발 관리 장치 제어 방법.

청구항 5
제4항에 있어서,상기 청소 요청을 전송할지 여부를 결정하는 단계는,상기 제1 모드로 동작하는 경우, 상기 적어도 하나의 로봇 청소기로 먼지 청소를 요청하는 것으로 결정하는 단계;상기 제2 모드로 동작하는 경우, 상기 적어도 하나의 로봇 청소기로 물걸레 청소를 요청하는 것으로 결정하는 단계; 및상기 제3 모드로 동작하는 경우, 상기 청소 요청을 전송하지 않는 것으로 결정하는, 신발 관리 장치 제어 방법.

청구항 6
제1항에 있어서,상기 청소 요청을 전송할지 여부를 결정하는 단계는,상기 신발의 오염도 정보 또는 상기 수행 중인 동작 모드 중 적어도 하나에 기초하여 상기 청소 요청을 전송하지 않기로 결정된 경우, 날씨 정보에 기초하여 청소 요청을 전송할지 여부를 결정하는 단계;미세 먼지 값이 기준 값 이상인 경우 먼지 청소를 요청하는 것으로 결정하는 단계; 및비 또는 눈이 오는 경우 물걸레 청소를 요청하는 것으로 결정하는 단계를 포함하는, 신발 관리 장치 제어 방법.

청구항 7
제1항에 있어서,상기 신발 관리 장치의 도어 열림 동작을 검출하는 단계;신발의 수납을 검출하는 단계;상기 신발 관리 장치의 문이 열리고 신발이 신규로 수납된 경우, 귀가 이벤트를 생성하는 단계; 및상기 귀가 이벤트를 서버로 전송하는 단계를 더 포함하는, 신발 관리 장치 제어 방법.

청구항 8
제7항에 있어서,상기 청소 요청을 전송할지 여부를 결정하는 단계는, 상기 귀가 이벤트가 검출된 것에 기초하여 수행되는, 신발 관리 장치 제어 방법.

청구항 9
제7항에 있어서,상기 귀가 이벤트에 기초하여, 상기 로봇 청소기로 라이다 센서 활성화 요청을 전송하는 단계를 더 포함하는 신발 관리 장치 제어 방법.

청구항 10
제7항에 있어서,상기 귀가 이벤트에 기초하여, 상기 서버로 상기 서버와 통신하는 적어도 하나의 IoT(Internet of Things) 장치의 사용 이벤트 정보의 수집을 요청하는 단계; 및상기 서버로 상기 적어도 하나의 IoT 장치의 사용 이벤트 정보에 기초하여 사용자의 동선 정보의 획득을 요청하는 단계를 더 포함하는 신발 관리 장치 제어 방법.

청구항 11
제10항에 있어서,상기 청소 요청을 전송하지 않는 것으로 결정된 경우, 상기 서버로 상기 획득된 사용자의 동선 정보의 삭제를 요청하는 단계를 더 포함하는 신발 관리 장치 제어 방법.

청구항 12
제10항에 있어서,상기 청소 요청을 전송하는 것으로 결정된 경우, 상기 서버로 상기 사용자의 동선 정보를 상기 적어도 하나의 로봇 청소기로 전송한 후에, 상기 사용자의 동선 정보를 삭제할 것을 요청하는 단계를 더 포함하는 신발 관리 장치 제어 방법.

청구항 13
제7항에 있어서,상기 귀가 이벤트에 기초하여 상기 청소 요청을 전송하기로 결정되고, 상기 청소 요청을 수신할 로봇 청소기 또는 상기 서버에 의해 사용자의 동선 정보를 생성할 수 없는 경우, 상기 청소 요청을 수신할 로봇 청소기로 미리 결정된 영역을 청소할 것을 요청하는 단계를 더 포함하는, 신발 관리 장치 제어 방법.

청구항 14
적어도 하나의 신발을 수납하는 컨테이너;상기 컨테이너 내부에 수납된 신발의 오염도를 측정하는 적어도 하나의 센서;상기 컨테이너 내부의 상기 적어도 하나의 신발의 상태를 관리하는 신발 관리 모듈;통신 모듈;적어도 하나의 인스트럭션을 저장하는 메모리; 및상기 메모리에 연결된 적어도 하나의 프로세서를 포함하고,상기 적어도 하나의 프로세서는, 상기 적어도 하나의 인스트럭션을 실행함에 의해,상기 적어도 하나의 센서의 측정 값을 이용하여 신발의 오염도 정보를 획득하고,복수의 동작 모드 중 하나의 동작 모드로 신발 관리 동작을 수행하도록 상기 신발 관리 모듈을 제어하고,상기 신발의 오염도 정보 또는 상기 수행 중인 동작 모드 중 적어도 하나에 기초하여, 적어도 하나의 로봇 청소기로 청소 요청을 전송할지 여부를 결정하고,상기 청소 요청을 전송하는 것으로 결정된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 청소 요청을 전송하도록 상기 통신 모듈을 제어하는, 신발 관리 장치.

청구항 15
제14항에 있어서,상기 적어도 하나의 센서는, 먼지 센서 및 습기 센서를 포함하고,상기 적어도 하나의 프로세서는 상기 적어도 하나의 인스트럭션을 실행함에 의해,상기 먼지 센서를 이용하여 먼지 오염도 정보를 획득하고,상기 습기 센서를 이용하여 습기 오염도 정보를 획득하는, 신발 관리 장치.

청구항 16
제15항에 있어서,제2항에 있어서,상기 적어도 하나의 프로세서는 상기 적어도 하나의 인스트럭션을 실행함에 의해,상기 먼지 오염도 정보가 먼지 기준 값 이상이라고 판단된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 먼지 청소를 요청하는 것으로 결정하고,상기 습기 오염도 정보가 습기 기준 값 이상이라고 판단된 것에 기초하여, 상기 적어도 하나의 로봇 청소기로 물걸레 청소를 요청하는 것으로 결정하는, 신발 관리 장치.

청구항 17
제14항에 있어서,상기 복수의 동작 모드는, 신발의 냄새 및 먼지를 제거하고 습기를 제거하는 제1 모드, 젖은 신발의 습기를 제거하는 제2 모드, 및 외출 전 신발을 관리하는 제3 모드를 포함하는, 신발 관리 장치.

청구항 18
제17항에 있어서,상기 적어도 하나의 프로세서는 상기 적어도 하나의 인스트럭션을 실행함에 의해,상기 제1 모드로 동작하는 경우, 상기 적어도 하나의 로봇 청소기로 먼지 청소를 요청하는 것으로 결정하고,상기 제2 모드로 동작하는 경우, 상기 적어도 하나의 로봇 청소기로 물걸레 청소를 요청하는 것으로 결정하고,상기 제3 모드로 동작하는 경우, 상기 청소 요청을 전송하지 않는 것으로 결정하는, 신발 관리 장치.
